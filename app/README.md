## PixiJS v5. Flexible Buttons

Flexible UI buttons in PixiJS v5. Quick, simple and common solution for PixiJS UI without ReactJS, VueJS and similar heavy libs.

Install dependencies:
```
npm i
```

Run on http://localhost:3000:
```
npm start
```

#### Building

Development build:
```
npm run build
```

Production build:
```
npm run production
```
